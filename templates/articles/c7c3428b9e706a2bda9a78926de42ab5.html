后端传值换行符至前端无法识别的问题

作者: wlmmmm                              创建时间:2023/07/17/ 20:48:01

问题：

后台传输文本变量（包含换行符<br>）至前端时，HTML页面未能识别<br>换行符，导致文本未换行，

原因：

后台传值时，br两侧的<>被转义为字符串&lt;和&gt;，前端将其识别为字符串，而非识别为HTML标签，导致换行失败。

解决方法：

将传值到前台的文本赋予innerHTML属性(通过javascript)，让浏览器能够识别文本中的HTML标签


<html>

<body>

  <div id='example'>

    <!-- 后台传入文本值 -->

    {{ textarea }}

  </div>

</body>

</html>

<script type="text/javascript">

  function modifyinnerHTML() {

    var text = document.getElementById('example') 

    text.innerHTML = text.innerText // 把该元素做为HTML元素来解析

  }

  modifyinnerHTML()

</script>

参考：

innerHTML：可以将获取的元素作为HTML元素进行解析或修改

innerText：可以将获取的元素作为纯文本进行解析或修改
